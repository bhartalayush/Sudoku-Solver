## Problem Description

Sudoku is a logic based number puzzle played on a 9 × 9 grid. The objective is to fill the grid with numbers from 1 to 9 such that each number appears only once in every row, every column, and each 3 × 3 subgrid.

Solving Sudoku puzzles manually requires logical reasoning and can be time consuming. This project develops a program that can automatically generate a Sudoku puzzle and compute its solution. The system first generates a valid completed Sudoku board, removes selected numbers to create a puzzle, and then applies algorithmic techniques to solve the puzzle while maintaining Sudoku constraints.

The project demonstrates how algorithm design and data structures can be used to solve constraint based problems like Sudoku.

## Data Structures Used

• Two dimensional array (9 × 9 integer matrix) to represent the Sudoku board
• Random number generation for creating different Sudoku puzzles
• Recursion for solving the puzzle

## Expected Features

• Automatic generation of a valid Sudoku puzzle
• Automatic solving of the puzzle
• Validation of Sudoku rules for rows, columns, and 3 × 3 subgrids
• Display of both the generated puzzle and the solved board and basic GUI
